Ορολογία


Αγγλικά Ελληνικά Περιγραφή
Starvation Ασιτία Κατάσταση όπου μια διαδικασία δεν μπορεί να αποκτήσει τους απαιτούμενους πόρους για να συνεχίσει την εκτέλεσή της λόγω άλλων διαδικασιών.
Race Condition Συνθήκη Αγώνα Κατάσταση όπου το αποτέλεσμα της εκτέλεσης εξαρτάται από την αλληλουχία της εκτέλεσης πολλών διεργασιών ή νημάτων.
Memory Management Διαχείριση Μνήμης Η διαδικασία διαχείρισης της μνήμης RAM για την αποδοτική εκτέλεση των διαδικασιών.
Process Scheduling Προγραμματισμός Διαδικασιών Η διαδικασία καθορισμού ποια διαδικασία θα εκτελείται και για πόσο χρόνο.
Virtualization Εικονικοποίηση Η διαδικασία δημιουργίας εικονικών εκδόσεων των πόρων του υπολογιστή, όπως οι εικονικές μηχανές.
Hypervisor Υπερβατήρας Λογισμικό που διαχειρίζεται τις εικονικές μηχανές και παρέχει την εικονικοποίηση του υλικού.
File Descriptor Περιγραφέας Αρχείου Μοναδικός αριθμός που αναγνωρίζει ένα ανοιχτό αρχείο ή πόρο στο λειτουργικό σύστημα.
Shared Memory Κοινή Μνήμη Μηχανισμός που επιτρέπει σε δύο ή περισσότερες διαδικασίες να έχουν πρόσβαση στο ίδιο τμήμα μνήμης.
Semaphore Σημαφόρος Μηχανισμός συγχρονισμού για τον έλεγχο της πρόσβασης σε κοινό πόρο από πολλές διαδικασίες.
Mutex Αποκλειστής Μηχανισμός συγχρονισμού που εξασφαλίζει ότι μόνο μία διαδικασία έχει πρόσβαση σε έναν κοινό πόρο τη φορά.
Daemon Δαίμονας Μια διαδικασία που τρέχει στο παρασκήνιο και παρέχει υπηρεσίες στο λειτουργικό σύστημα.
Kernel Mode Λειτουργία Πυρήνα Λειτουργία λειτουργικού συστήματος με πλήρη πρόσβαση σε όλο το υλικό του υπολογιστή.
User Mode Λειτουργία Χρήστη Λειτουργία που περιορίζει την πρόσβαση των εφαρμογών στο υλικό του υπολογιστή για λόγους ασφάλειας.
System Resource Πόρος Συστήματος Στοιχείο του υπολογιστή, όπως η CPU ή η μνήμη, που χρησιμοποιείται από το λειτουργικό σύστημα και τις εφαρμογές.
Service Υπηρεσία Διεργασία ή πρόγραμμα που εκτελεί μια συγκεκριμένη λειτουργία και συνήθως τρέχει στο παρασκήνιο.
Process Isolation Απομόνωση Διαδικασιών Η διαδικασία που εξασφαλίζει ότι οι διαδικασίες δεν αλληλεπιδρούν αρνητικά μεταξύ τους.
Context Συμφραζόμενα Η κατάσταση και οι πληροφορίες που σχετίζονται με την εκτέλεση μιας διαδικασίας ή νήματος.
Swap Space Χώρος Ανταλλαγής Τμήμα του δίσκου που χρησιμοποιείται για την προσωρινή αποθήκευση δεδομένων που δεν χωρούν στη RAM.
Kernel Space Χώρος Πυρήνα Το τμήμα της μνήμης RAM που χρησιμοποιείται από τον πυρήνα του λειτουργικού συστήματος.
User Space Χώρος Χρήστη Το τμήμα της μνήμης RAM που χρησιμοποιείται από τις εφαρμογές και τους χρήστες.
Executable Εκτελέσιμο Αρχείο που περιέχει κώδικα που μπορεί να εκτελείται από τον υπολογιστή.
Library Βιβλιοθήκη Σύνολο προκαθορισμένων λειτουργιών ή ρουτινών που μπορούν να χρησιμοποιηθούν από προγράμματα.
Linker Συνδετήρας Λογισμικό που συνδυάζει διάφορα κομμάτια κώδικα και βιβλιοθήκες σε ένα ενιαίο εκτελέσιμο αρχείο.
Loader Φορτωτής Λογισμικό που φορτώνει εκτελέσιμα αρχεία στη μνήμη για εκτέλεση.
Bootloader Φορτωτής Εκκίνησης Μικρό πρόγραμμα που φορτώνει το λειτουργικό σύστημα κατά την εκκίνηση του υπολογιστή.
System Call Interface Διεπαφή Κλήσεων Συστήματος Η διεπαφή που επιτρέπει στις εφαρμογές να κάνουν κλήσεις στο λειτουργικό σύστημα.
Memory Leak Διαρροή Μνήμης Κατάσταση όπου μια εφαρμογή χρησιμοποιεί μνήμη χωρίς να την απελευθερώνει, προκαλώντας εξάντληση της διαθέσιμης μνήμης.
Resource Allocation Κατανομή Πόρων Η διαδικασία διαχείρισης και κατανομής πόρων όπως η CPU, η μνήμη και οι συσκευές.
Process Synchronization Συγχρονισμός Διαδικασιών Η διαδικασία εξασφάλισης της σωστής αλληλουχίας εκτέλεσης διαδικασιών για την αποφυγή συγκρούσεων.
Resource Deadlock Αδιέξοδο Πόρων Κατάσταση όπου η πρόσβαση σε πόρους προκαλεί αδιέξοδο μεταξύ διαδικασιών.
Privilege Escalation Κλιμάκωση Δικαιωμάτων Η διαδικασία κατά την οποία μια εφαρμογή ή χρήστης αποκτά υψηλότερα δικαιώματα από αυτά που αρχικά είχαν.
File Descriptor Table Πίνακας Περιγραφέων Αρχείων Δομή δεδομένων που διατηρεί πληροφορίες για τα ανοιχτά αρχεία και πόρους.
Signal Σήμα Μηχανισμός επικοινωνίας μεταξύ διεργασιών ή μεταξύ διεργασιών και του λειτουργικού συστήματος.
Thread Synchronization Συγχρονισμός Νημάτων Η διαδικασία διαχείρισης της συγχρονισμένης εκτέλεσης πολλών νημάτων σε μια διαδικασία.
Task Scheduler Προγραμματιστής Εργασιών Λογισμικό που καθορίζει την σειρά και την προτεραιότητα εκτέλεσης των εργασιών ή διεργασιών.
System Monitor Παρακολούθηση Συστήματος Εργαλείο που παρακολουθεί τη χρήση πόρων του υπολογιστή και τη συμπεριφορά του λειτουργικού συστήματος.
Thread Pool Πισίνα Νημάτων Σύνολο προ-δημιουργημένων νημάτων που μπορούν να χρησιμοποιηθούν από εφαρμογές για την εκτέλεση εργασιών.
Service Manager Διαχειριστής Υπηρεσιών Λογισμικό που διαχειρίζεται και ελέγχει τις υπηρεσίες που εκτελούνται στο λειτουργικό σύστημα.
Process State Κατάσταση Διαδικασίας Η τρέχουσα κατάσταση μιας διαδικασίας, όπως εκτέλεση, αναστολή, ή τερματισμός.
Atomic Operation Ατομική Λειτουργία Λειτουργία που εκτελείται ως ενιαία, αδιαίρετη ενέργεια χωρίς παρεμβολές.
File Access Control Έλεγχος Πρόσβασης Αρχείων Μηχανισμός που περιορίζει την πρόσβαση σε αρχεία βάσει δικαιωμάτων χρηστών.
Job Scheduling Προγραμματισμός Εργασιών Η διαδικασία προγραμματισμού των εργασιών για την εκτέλεση σε καθορισμένα χρονικά διαστήματα.
Service Request Αίτημα Υπηρεσίας Αίτημα για την εκτέλεση μιας συγκεκριμένης υπηρεσίας ή λειτουργίας από το λειτουργικό σύστημα.
Service Daemon Δαίμονας Υπηρεσίας Διαδικασία που εκτελείται στο παρασκήνιο και παρέχει υπηρεσίες σε άλλες διεργασίες ή χρήστες.
Resource Manager Διαχειριστής Πόρων Συστατικό του λειτουργικού συστήματος που διαχειρίζεται τη διανομή και χρήση των πόρων του υπολογιστή.
Execution Context Συμφραζόμενα Εκτέλεσης Συνόλο δεδομένων και καταστάσεων που σχετίζονται με την εκτέλεση μιας διαδικασίας ή νήματος.
System Call Interface Διεπαφή Κλήσεων Συστήματος Η διεπαφή μέσω της οποίας οι εφαρμογές καλούν λειτουργίες του λειτουργικού συστήματος.
System Performance Απόδοση Συστήματος Η συνολική απόδοση του υπολογιστικού συστήματος, περιλαμβάνοντας ταχύτητα και αποδοτικότητα.
System Resource Management Διαχείριση Πόρων Συστήματος Η διαδικασία κατανομής και διαχείρισης πόρων του υπολογιστή για τις διάφορες διαδικασίες και εφαρμογές.
Process Control Block (PCB) Μπλοκ Ελέγχου Διαδικασίας (PCB) Δομή δεδομένων που περιέχει τις πληροφορίες κατάστασης μιας διαδικασίας.
Thread Control Block (TCB) Μπλοκ Ελέγχου Νήματος (TCB) Δομή δεδομένων που περιέχει τις πληροφορίες κατάστασης ενός νήματος.
Resource Allocation Graph Γράφος Κατανομής Πόρων Γράφος που απεικονίζει την κατανομή και τις απαιτήσεις πόρων μεταξύ διαδικασιών.
Task Management Διαχείριση Εργασιών Η διαδικασία παρακολούθησης και ελέγχου των εργασιών που εκτελούνται στο σύστημα.
Application Programming Interface (API) Διεπαφή Προγραμματισμού Εφαρμογών (API) Σύνολο εργαλείων και πρωτοκόλλων που επιτρέπουν σε εφαρμογές να αλληλεπιδρούν μεταξύ τους.
System Uptime Χρόνος Λειτουργίας Συστήματος Η χρονική περίοδος κατά την οποία ένα σύστημα είναι ενεργό και λειτουργεί χωρίς διακοπές.
System Shutdown Απενεργοποίηση Συστήματος Η διαδικασία τερματισμού της λειτουργίας του υπολογιστή με ασφαλή τρόπο.
Load Balancing Εξισορρόπηση Φορτίου Η διαδικασία κατανομής φορτίου ή εργασιών σε πολλούς πόρους για βελτίωση απόδοσης και αξιοπιστίας.
CacheΜνήμη Κρυφής ΑποθήκευσηςΜια περιοχή αποθήκευσης που αποθηκεύει δεδομένα για ταχύτερη πρόσβαση και μείωση του χρόνου φόρτωσης.
UI (User Interface)Διεπαφή ΧρήστηΗ συνολική εμπειρία και η αλληλεπίδραση του χρήστη με μια εφαρμογή ή σύστημα.
UX (User Experience)Εμπειρία ΧρήστηΗ συνολική αίσθηση και ικανοποίηση του χρήστη κατά την αλληλεπίδραση με ένα προϊόν ή υπηρεσία.
Systems IntegrationΕνοποίηση ΣυστημάτωνΗ διαδικασία συνένωσης διαφορετικών υπολογιστικών συστημάτων και εφαρμογών ώστε να λειτουργούν ως ενιαία λύση.
Swap Spac (SWAP)eΠεριοχή στον σκληρό δίσκο που χρησιμοποιείται ως εικονική μνήμη για να αυξήσει τη διαθέσιμη μνήμη RAM
