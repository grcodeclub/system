Αρμοδιότητες

  1. Να λειτουργεί ως ενδιάμεσος, αναμεσά στον χρήστη και της μηχανής.
  2. Να διαχειρίζεται τις δυνατότητες και τους Η/Υ
  3. Να μεταφέρει εντολές ή/και τις απαιτήσεις του χρήστη στον Η/Υ
  4. Να δίνει χρήσιμες πληροφορίες για την κατάσταση του συστήματος
  5. Να διαχειρίζεται την RAM
  6. Να ενεργοποιεί και να δίνει οδηγίες στην ΚΜΕ
  7. Να διαχειρίζεται τις συσκευές εισόδου / εξόδου
  8. Να οργανώνει και να διαχειρίζεται τα αρχεία του συστήματος
  9. Να ελέγχει την εκτέλεση των προγραμμάτων
  10. Να εφαρμόζει μηχανισμού που βελτιώνουν την ασφάλεια του Η/Υ από διάφορους κίνδυνους
